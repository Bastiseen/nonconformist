nonconformist.nc.RegressorNc¶

class nonconformist.nc.RegressorNc(model_class, err_func, inverse_err_func, model_params=None)¶

Nonconformity scorer using an underlying regression model.

Parameters:

model_class : class

The model_class should be implement the fit(x, y) and predict(x) methods, as used by the regression models present in the scikit-learn library.

err_func : callable

Scorer callable object with signature score(estimator, x, y).

inverse_error_func : callable

Inverse (or partial inverse) of err_func.

model_params : dict, optional

Dict containing keyword parameters to pass to model_class upon initialization.
